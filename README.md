# Gof 디자인 패턴

GoF(Gang of For)라는 4명의 저자가 정리한 23개의 디자인 패턴

크게 3개의 카테고리로 디자인 패턴을 나눌 수 있다.
- 객체 생성과 관련 있는 패턴
- 구조적인 패턴
- 행동과 관련 있는 패턴
